# IT Company Website

## Overview
This project is a website for an IT Company. It uses PHP and Composer for dependency management.

## Prerequisites
- PHP >= 5.5
- Composer

## Installation
1. Clone the repository.
2. Run the following command to install dependencies:
   ```bash
   composer install
   ```
